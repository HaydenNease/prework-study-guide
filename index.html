Git

The following is an example of the Bash commands used to follow the GitFlow of a project:

  1. git status checks what branch we are currently on
  2. checkout main switches to main branch
  3. git pull origin main pulls latest main branch
  4. git checkout -b feature/branch-name creates a new branch and switches to it
  5. code . opens Visual Studio Code to make changes
  6. git add -A adds the changed files in our working directory to the staging area of Git
  7. git commit -m "short descriptive message of the changes implemented" commits changes to our repository with message
  8. git pull origin main
  9. git push origin feature/branch-name pushes the changes to main

Additional Commands:

  • touch filename.filetype creates a new file
